Divs

Divs are containers for other elements. By default, they are invisible. However, together with CSS they are a powerful layouting and styling tool. Divs can be nested!

Here, we use different background colors to show how CSS style can be assigned to tags, classes, and ids. Not that the class can be used by different tags (e.g., one p and one div tag have the same class in this example).

Here, we already see the concept of specificity in CSS: The id selector is more specific than the class selector, and that one is more specific than the general element/tag selector. Also, the div at the very top has an inline style, which overrides the internal style (since it's "closer").
