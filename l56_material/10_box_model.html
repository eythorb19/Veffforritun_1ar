Margin

Margins are added outside the border area. Here, there is a 10px margin to the right side.

Width

The width is set to 60% here, so it automatically re-sizes when the window width changes.

Padding, box-sizing

Padding is visible (it gets the same style as the content). It's added "within" the element. Note that usually padding is not included in the width/height! So here it would be 60% + padding + border. The box-sizing property changes this to include both.
