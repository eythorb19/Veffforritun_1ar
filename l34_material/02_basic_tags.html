Basic HTML Tags (Content)

Here we present a number of basic HTML tags you might need for everyday use.

You can find a full reference and further information here.

Tag Reference:

These are a number of basic tags you will need:

  • <h1> to <h6>: Different headings.
  • <p>: Paragraphs.
  • <a>: Hyperlinks.
    1. href attribute: The target URL.
    2. target attribute: How to open the URL (in the same window, the parent, a new window, etc.).
  • <img>: Images.
    1. src attribute: The URL of the image to show (typically as a relative URL).
    2. alt attribute: An alternative text explaining the image.
  • <ol> or <ul>: Ordered or unordered lists.
  • <li>: Elements in a list.
  • One item
  • Another item
    1. A sub item
  • A third item
