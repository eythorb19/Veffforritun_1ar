HTML Hyperlinks

Here we present how hyperlinks work.

Relative links

You can link to documents on the same host (e.g., in the same folder) using relative links: This opens 03b_target.html

Absolute links

You can link to external resources (e.g., another website) using absolute links: This opens Wikipedia

Bookmarks

You can give elements in your page an id. Then you can use hyperlinks to directly jump there.
Absolute URL: https://en.wikipedia.org/wiki/HTML#Elements
Relative URL (here in the current doc): #bookmarks
